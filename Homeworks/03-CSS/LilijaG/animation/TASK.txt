ПОМЕСТИТЬ РЕЗУЛЬТАТ В СВОЮ ПАПКУ

-------------------------------------------------------------------------------------------
Попробуем сделать вот такую анимацию demo.gif
-------------------------------------------------------------------------------------------

Для начала надо создать контейнер флексбокс и в нём расположить 3 элемента. 
Для закругления можно использовать border-radius: 50%; тогда у нас получится круг при 
равном соотношении сторон.

Если не получается, можно использовать animation.html и animation.css для примера,
но попробуйте создать это сами.

-------------------------------------------------------------------------------------------
АНИМАЦИЯ
-------------------------------------------------------------------------------------------
Мы можем создать анимацию разнымы способами: 
1) увеличить значение width и height;
2) использовать transform и увеличить значение scale + изменить цвет.

Для каждого из 3-х элементов можно задействовать созданную анимацию.
Для этого необходимо задействовать несколько свойст:

animation-name,
animation-duration,
animation-delay,
animation-timing-function,

чтобы получить такой результат demo.gif

Попробуем сделать вот такую анимацию demo.gif
-------------------------------------------------------------------------------------------

Для начала надо создать контейнер флексбокс и в нём расположить 3 элемента 
Для закрушления можно использовать border-radius: 50%; тогда у нас получиться круг при равном соотнощении сторон

Если не получаеться можно использовать animation.html и animation.css для примера
но попробуйте создать это сами

-------------------------------------------------------------------------------------------
АНИМАЦИЯ
-------------------------------------------------------------------------------------------
Мы можем создать анимацию разнымы способами 
1) увеличить значение width and height 
2) использовать transform и увеличить значение scale
+ изменить цвет

для каждого из 3 элементов можно задействовать созданаю анимацию
необходимо задействовать несколько свойст 

animation-name
animation-duration
animation-delay
animation-timing-function

что бы получить такой результат demo.gif
